My first app!

Team Divergent - Stats

Created this team stats page for an Ultimate team. As assistant coach, I created this app to provide insight to the other coaches in order to determine starting order, player matching, and other decisions that would help the team get better.

Hosted via Heroku.

Language(s): Python, Plotly, Dash

Standalone Python Dash App

Using the pyinstaller module, I created a simple proof-of-concept application that runs as a .exe file downloaded from the site onto the users computer that can run on any computer, regardless of pythoin version or if it is even installed.

Runs on user's localhost:8050.

Language(s): Python, Plotly, Dash
